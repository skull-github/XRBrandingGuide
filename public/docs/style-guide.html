Typography Scale

Our typography system uses scaled-down sizes (50% of original design specs) for better web readability:

Score (70px)

Used for: Key metrics, large numbers

.text-score
Marquee (48px)

Used for: Main headlines, hero text

.text-marquee
Display (38px)

Used for: Section headers

.text-display
Title (29px)

Used for: Subsection headers

.text-title
Headline (24px)

Used for: Component headers

.text-headline
SubHeadline (19px)

Used for: Minor headers

.text-subheadline
Body (17px)

Used for: Main content text

.text-body
System (15px)

Used for: UI elements, labels

.text-system
Legal (13px)

Used for: Fine print, disclaimers

Usage Examples

HTML Usage

<!-- Main page title -->
<h1 class="text-marquee">MLB XR Branding Guide</h1>

<!-- Section headers -->
<h2 class="text-title-bold">Quick Start</h2>

<!-- Body content -->
<p class="text-body">This is the main content paragraph.</p>

<!-- Code snippets -->
<code class="text-code">npm install</code>

CSS Variables

/* Typography variables */
--font-marquee: 48px;
--font-title: 29px;
--font-body: 17px;

/* Usage in custom CSS */
.custom-header {
    font-size: var(--font-marquee);
    font-weight: var(--weight-bold);
}
